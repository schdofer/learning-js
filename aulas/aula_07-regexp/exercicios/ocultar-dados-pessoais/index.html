Ocultar Dados Pessoais

Você foi contratado para criar um programa que leia um texto contendo uma mensagem de boas vindas e uma lista de 20 classificados em um concurso. Esta lista tem nome, e-mail, cpf e data de nascimento..

Seu programa deve ler o texto, pesquisar por padrões de extração utilizando RegExp e ocultar os dados de E-Mail e CPF antes de apresentá-los na tela, garantindo assim requisitos da LGPD.
